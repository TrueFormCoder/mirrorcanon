# Public / Access Lane Assets

**Audience:** General public, first-time readers  
**Reading Grade Target:** 8–10  
**Purpose:** Invitation and comprehension without intimidation

This lane exists to reduce entry friction while preserving structural truth.

## Allowed
- Single, clear symbol per asset
- High whitespace
- Calm, neutral presentation
- Simple metaphors that map directly to structure
- Procedural Slate or Continuity Green accents only

## Prohibited
- Diagrams
- Dense geometry
- Multiple symbols
- Gold-dominant palettes
- Editorial caricature
- Figures that read as people
- Visual cleverness or irony

## Usage
- “Start Here” pages
- Intro explainers
- MirrorTone™ summaries
- Public onboarding sections

## Rule
If a reader cannot explain the image in one sentence without jargon, the asset does not belong in this lane.

All assets here are **Public-only** and may not be reused in Scholar or Institutional contexts without redesign.

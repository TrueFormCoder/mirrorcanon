# Mainstream / Press Lane Assets

**Audience:** Journalists, editors, cultural interpreters  
**Reading Grade Target:** 11â€“12  
**Purpose:** Interpretation control and framing

This lane exists to guide press understanding before narrative collapse.

## Allowed
- Editorial illustration (New Yorker / Economist lineage)
- Figurative but abstracted forms
- Clear power geometry
- Muted palette
- Oxblood or Procedural Slate accent only

## Prohibited
- Satire or humor
- Celebrity likeness or recognizability
- Violence
- Logos or brand marks
- Meme aesthetics
- Use inside Canon or Institutional documents

## Usage
- Issue hero images
- Press kits
- Social previews
- Media embeds

## Rule
If an editor could misread this as commentary or satire, the asset fails.

Assets in this lane are **visual-only** and must always be paired with press-context copy.

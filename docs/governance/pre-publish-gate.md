# MirrorCanon™ Pre-Publish Gate (v1.0)

**Status:** Active  
**Purpose:** Deterministic release process for Canon artifacts

---

## Step 1 — Run Canon Lint

Run the Canon Lint Gate against the full artifact.

- If any required check fails, rewrite.
- Do not publish partial compliance.

---

## Step 2 — Run Red-Flag Scan

Scan for red-flag tokens.

- If present, remove or justify structurally.
- No aesthetic language allowed in Canon.

---

## Step 3 — Archive Metadata

Ensure archive copy includes:

- Lane
- Reading grade
- Version
- Date
- Author of record

Public threads exclude metadata.

---

## Step 4 — Final Tests

Confirm:

- hostile excerpt test
- institution test
- translation test
- minute test

If any test fails, rewrite.

---

## Step 5 — Index Update

Update Canon Index with:

- paragraph IDs
- kernel links (if affected)
- diagnostic links (if affected)

---

## Versioning

**v1.0** · Promoted into Canon governance  
Updates require explicit version bump and rationale.

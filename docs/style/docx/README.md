# DOCX Style Sources

This folder contains **DOCX source files** for style documentation.

## Purpose
- Text-first editing
- Review cycles and markup
- Source for layout-ready conversions

## Rules
- DOCX files are sources, not distribution artifacts
- Export PDFs to `docs/style/pdf`
- Keep version numbers in filenames
- Do not overwrite existing versions

## Naming Convention
MirrorCanon_[ArtifactName]_vX.Y.docx

Example:
MirrorCanon_Editor_OnePage_Guide_v1.3.docx

# Editable Style Assets

This folder contains **layout source files** for MirrorCanon™ style and governance documents.

## Structure
- `pages/` → primary design files (Apple Pages)
- `docx/` → cross-platform editable backups

## Rules
- Do not edit PDFs directly
- Pages files are the preferred source when available
- DOCX files must match the latest Pages version
- Increment version numbers on any layout change

Editable files control formatting, spacing, and visual hierarchy.,
